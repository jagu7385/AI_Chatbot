🤖 AI Chatbot

A simple AI chatbot that can answer user queries using Natural Language Processing techniques.

🚀 Features
Responds to user questions
Uses NLP for understanding text
Simple and interactive interface
🛠️ Technologies Used
Python
NLTK / spaCy
Scikit-learn (optional)
▶️ How to Run
Clone the repository
Install required libraries
pip install nltk spacy scikit-learn
Run the chatbot
python chatbot.py
